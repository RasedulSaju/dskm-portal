PART 6: DEPLOYMENT INSTRUCTIONS

════════════════════════════════════════════════════════════════════════════

FOR LOCAL TESTING (XAMPP):

Step 1: Install XAMPP
  • Download from: https://www.apachefriends.org/
  • Choose PHP 8.1+ version
  • Install with Apache and MySQL selected

Step 2: Extract Project
  • Extract dskm-portal folder to: C:\xampp\htdocs\
  
Step 3: Create Database
  • Open: http://localhost/phpmyadmin
  • Login: root / (empty password)
  • Create database: dskm_portal
  • Select: utf8mb4_unicode_ci

Step 4: Import Schema
  • In phpMyAdmin, select dskm_portal
  • Click "Import" tab
  • Choose: database/schema.sql
  • Click "Go"
  • Wait for completion message

Step 5: Configure .env
  • Copy: .env.example → .env
  • Edit .env:
    APP_ENV=local
    APP_DEBUG=true
    DB_HOST=localhost
    DB_DATABASE=dskm_portal
    DB_USERNAME=root
    DB_PASSWORD=

Step 6: Start Services
  • Open XAMPP Control Panel
  • Start Apache (should be green)
  • Start MySQL (should be green)

Step 7: Access Application
  • Open browser
  • Go to: http://localhost/dskm-portal/public
  • You should see landing page

Step 8: Login
  • Click "Login"
  • Username: superadmin
  • Password: Admin@1234
  • You should see dashboard

✅ If all steps work → Application is ready


FOR LARAGON SETUP:

Step 1: Install Laragon
  • Download from: https://laragon.org/
  • Run installer
  • Choose installation location

Step 2: Extract Project
  • Extract dskm-portal to: C:\laragon\www\

Step 3: Start Services
  • Click "Start All" in Laragon
  • Wait for Apache and MySQL to be green

Step 4: Create Database
  • In Laragon menu: Click "Database"
  • Click "MySQL Client"
  • Run commands:
    CREATE DATABASE dskm_portal CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
    USE dskm_portal;
    SOURCE C:/laragon/www/dskm-portal/database/schema.sql;

Step 5: Configure .env
  • Copy: .env.example → .env
  • Edit .env:
    APP_ENV=local
    APP_DEBUG=true
    DB_HOST=127.0.0.1
    DB_DATABASE=dskm_portal
    DB_USERNAME=root
    DB_PASSWORD=

Step 6: Access Application
  • In Laragon, right-click project
  • Select "Open in Browser"
  • Or: http://dskm-portal.local

Step 7: Login
  • Username: superadmin
  • Password: Admin@1234

✅ If all steps work → Application is ready

PART 8: BACKUP & RESTORE GUIDE

════════════════════════════════════════════════════════════════════════════

BACKUP PROCEDURE:

Database Backup:
  # Command:
  mysqldump -u root -p dskm_portal > backup_$(date +%Y%m%d).sql

Files Backup:
  # Command:
  tar -czf dskm_backup_$(date +%Y%m%d).tar.gz /path/to/dskm-portal/

Full Backup Script:
  #!/bin/bash
  BACKUP_DIR="/backups/dskm"
  DATE=$(date +%Y%m%d_%H%M%S)
  
  mkdir -p $BACKUP_DIR
  
  # Database
  mysqldump -u root -p dskm_portal | gzip > $BACKUP_DIR/dskm_db_$DATE.sql.gz
  
  # Files
  tar -czf $BACKUP_DIR/dskm_files_$DATE.tar.gz /var/www/dskm-portal/
  
  # Keep only 30 days
  find $BACKUP_DIR -name "*.gz" -mtime +30 -delete


RESTORE PROCEDURE:

Database Restore:
  # Command:
  mysql -u root -p dskm_portal < backup_20260526.sql

Files Restore:
  # Command:
  tar -xzf dskm_backup_20260526.tar.gz -C /path/to/


AUTOMATED BACKUP (Cron):

Add to crontab:
  0 2 * * * /path/to/backup_script.sh

This runs daily at 2 AM
