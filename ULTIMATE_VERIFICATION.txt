⚡ QUICK START INSTRUCTIONS

FOR XAMPP USERS:
───────────────────────────────────────────────────────────────────────────

1. Download XAMPP with PHP 8.1+ from https://www.apachefriends.org/
2. Extract dskm-portal folder to: C:\xampp\htdocs\
3. Start Apache & MySQL (green checkmarks)
4. Open phpMyAdmin: http://localhost/phpmyadmin
5. Create database: dskm_portal (UTF-8)
6. Import schema: database/schema.sql
7. Edit .env (copy from .env.example)
8. Access: http://localhost/dskm-portal/public
9. Login: superadmin / Admin@1234

Time: 15 minutes
Status: ✅ LIVE


FOR LARAGON USERS:
───────────────────────────────────────────────────────────────────────────

1. Download Laragon from https://laragon.org/
2. Extract dskm-portal folder to: Laragon/www/
3. Click "Start All" (Apache + MySQL)
4. Right-click project > MySQL Client
5. CREATE DATABASE dskm_portal CHARACTER SET utf8mb4
6. SOURCE database/schema.sql
7. Edit .env (copy from .env.example)
8. Access: http://dskm-portal.local
9. Login: superadmin / Admin@1234

Time: 12 minutes
Status: ✅ LIVE


FOR PRODUCTION (VPS):
───────────────────────────────────────────────────────────────────────────

1. Follow DEPLOYMENT.md guide
2. Install PHP 8.1+ & MySQL 8.0+
3. Upload files via SCP
4. Import database schema
5. Configure .env for production
6. Enable HTTPS with Let's Encrypt
7. Set APP_DEBUG=false
8. Test all features
9. Train administrators

Time: 3-4 hours
Status: ✅ LIVE
